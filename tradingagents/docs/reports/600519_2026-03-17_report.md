# 九、从过去经验中学习：市场在贵州茅台这类“神股”上最容易犯什么错？

- 从高增长资产，变成稳健型资产；
- 从讲成长溢价，变成讲分红和防御；
- 从高弹性收益，变成低波动、低回报预期。

如果你还按成长股的收益预期去买，就容易失望。

## 教训4：缺数据时应更保守，而不是更浪漫
现在连最新技术面、量价关系、支撑压力都没有。  
这种时候最合理的态度应该是：
- 降低确定性表达；
- 等待更完整验证；
- 审视赔率，而不是沉迷叙事。

而不是因为“公司伟大”就自动站到看多一边。
